class SparseGraphRow¶

A lightweight view into a range of elements in a vector.

The SparseGraphRow class provides a lightweight view into a range of elements in a vector. The class is used to represent a row of non-zero elements in a matrix stored as a compressed sparse row (CSR) format. A SparseGraphRow object consists of a reference to the vector of non-zero elements, along with start and end indices that delimit the range of elements in the row.

The SparseGraphRow class provides two public member functions: size() and operator[](). The size() function returns the number of non-zero elements in the row, and the operator[]() function returns the value at a specific index in the row.
